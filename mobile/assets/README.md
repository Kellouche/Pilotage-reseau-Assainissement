# Assets requis pour l'application mobile

Les fichiers suivants doivent être créés dans le dossier `assets/` :

## Images requises (PNG, 1024x1024px minimum)
- `icon.png` - Icône principale de l'application
- `splash.png` - Image de splash screen
- `adaptive-icon.png` - Icône adaptative pour Android

## Comment créer ces assets :

1. **Icône principale** : Créez un logo SWMM ou un symbole d'assainissement
2. **Splash screen** : Image de fond pour l'écran de démarrage
3. **Icône adaptative** : Version Android de l'icône (avec marges)

Vous pouvez utiliser des outils comme :
- Canva, Figma, ou GIMP pour créer les images
- Des générateurs d'icônes en ligne
- Ou simplement utiliser des placeholders temporaires

Pour le développement, l'application fonctionnera même sans ces assets (Expo utilisera des defaults).
